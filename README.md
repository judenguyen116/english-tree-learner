# English Tree Learner

Chrome Extension giúp học từ vựng tiếng Anh ngay trong lúc đọc báo.

## Thư mục
- `extension/etl-demo-dict-v2/`: phiên bản tốt nhất hiện tại (đã fix context + dictionary)
- `extension/etl-demo-dict/`: bản có dictionary (v1)
- `extension/etl-demo-save/`: bản Save cơ bản (v0.2)
- `releases/*.zip`: gói sẵn để upload store
- `docs/English_Tree_Learner_Design.docx`: tài liệu thiết kế

## Cách chạy local
1. `chrome://extensions` → bật **Developer mode**
2. **Load unpacked** → chọn một trong các thư mục con trong `extension/`
3. Mở `demo-article.html` trong thư mục đã load để test

## Build zip
Dùng các file có sẵn trong `releases/`, hoặc zip lại thư mục tương ứng.
